# GooseBattery

GooseBattery is a free and open-source macOS battery monitoring app built with SwiftUI.

It gives you a clean live dashboard for tracking how your Mac battery is performing in real time, with detailed battery data that is usually hidden or hard to read.

## What GooseBattery Does

- Shows realtime battery charging speed
- Shows realtime battery consumption speed
- Displays battery health
- Displays filled battery capacity in mAh
- Displays empty battery capacity in mAh
- Displays current full-charge battery capacity
- Displays original design battery capacity
- Shows adapter wattage when available
- Shows live battery power
- Shows cycle count
- Shows battery temperature
- Shows battery percentage in a large live dashboard
- Shows continuous battery flow history with a graph
- Provides a live menu bar widget for quick access
- Lets users open the full dashboard from the menu bar
- Lets users enable or disable the menu bar widget
- Includes custom goose branding, logo, and app icon
- Includes an animated goose-themed dashboard experience

## Main Features

- Realtime charging speed in `mAh/h`
- Realtime battery drain in `mAh/h`
- Battery health calculated from current capacity vs design capacity
- Raw battery values in `mAh` for better transparency
- Menu bar battery widget
- Full desktop dashboard for deeper monitoring
- Clean modern macOS interface
- Free to use for everyone
- Open-source and community-friendly

## Battery Data Shown

- Battery percentage
- Charging status
- AC power connected status
- Charging speed
- Consumption speed
- Net battery flow
- Filled battery capacity
- Empty battery capacity
- Maximum current capacity
- Design capacity
- Battery health percentage
- Adapter input wattage
- Battery power in watts
- Cycle count
- Battery temperature

## Menu Bar Widget

- Displays live battery status directly in the macOS menu bar
- Shows quick battery information without opening the full app
- Opens the dashboard instantly
- Can be turned on or off from inside the app

## Dashboard Experience

- Large live battery percentage display
- Continuous battery flow chart
- Easy-to-read metric cards
- Fast one-second refresh updates
- Animated goose branding for a more memorable and playful UI

## Why GooseBattery

- Many battery apps are paid, limited, or hide raw battery values
- GooseBattery is built to be simple, transparent, and fully free
- It is useful for developers, power users, laptop owners, and anyone who wants to understand Mac battery behavior better

## Built For

- macOS users
- MacBook users who want realtime battery data
- Users who want battery health and charging insights
- Users who prefer free and open-source software

## Open Source Mission

GooseBattery is made to give everyone access to a beautiful, transparent, and fully free macOS battery monitor with realtime metrics, menu bar convenience, and clear battery health information.
