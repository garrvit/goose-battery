# GooseBattery

GooseBattery is a free and open-source macOS battery monitor that shows realtime battery charging speed, battery consumption, battery health, adapter input wattage, and filled/empty battery capacity in mAh.

Built with SwiftUI for macOS, GooseBattery reads live battery data directly from the system and presents it in a clean dashboard with continuous updates. It is designed for anyone who wants a simple, transparent way to monitor battery performance and charging behavior on Mac.

## Features

- Realtime charging speed in mAh/h
- Realtime battery consumption in mAh/h
- Battery health based on current full-charge capacity vs design capacity
- Filled battery capacity in mAh
- Empty battery capacity in mAh
- Adapter wattage detection
- Live battery power, cycle count, and temperature
- Free to use for everyone on macOS

## Why GooseBattery?

Most battery apps hide raw data or lock useful features behind a paywall. GooseBattery is made to be simple, useful, and fully free for the macOS community.
